Utiliser un Framework

En programmation informatique le framework est un ensemble de composants logiciels structurels qui sert à créer tout ou juste une partie d'un logiciel. Il nous simplifie les choses en donnant des composants faciles a utiliser.

Les Framework sont tous bases sur des architecture logicielle destiné aux interfaces graphiques

MVC

Modèle-vue-contrôleur ou MVC est un motif d'architecture logicielle destiné aux interfaces graphiques, lancé en 1978 et très populaire pour les applications web. Le motif est composé de trois types de modules ayant trois responsabilités différentes : les modèles, les vues et les contrôleurs.

  • Un modèle (Model) contient les données à afficher ;
  • Une vue (View) contient la présentation de l'interface graphique ;
  • Un contrôleur (Controller) contient la logique concernant les actions effectuées par l'utilisateur.

    • MVVM

    Dans le patron MVVM il y a une communication bidirectionnelle entre la vue et le modèle, les actions de l'utilisateur entraînent des modifications des données du modèle

    MVP

    Dans le patron MVP, le contrôleur est remplacé par une présentation. La présentation est créée par la vue et lui est associée par une interface. Les actions utilisateur déclenchent des événements sur la vue, et ces événements sont propagés à la présentation en utilisant l'interface.

    Framework JavaScript

    Un framework JavaScript est une bibliothèque de fonctions pré-écrites en JavaScript facilitant le développement d'applications fondées sur JavaScript, et en particulier pour Ajax et autres applications centrées Web.

    J'ai appris a me servir, durant mon stage, de vue.js un framework JS monpages. Dans un fichier vue il y a trois composants un pour l'HTML, un pour le CSS et un autre pour le JS. C'est d'ailleurs pour ça que le framework est surtout utilisé. Il est utilisé pour sa simplicité et sa facilité de maintenance. C'est d'ailleurs grace a ce Framework que j'ai pu en apprendre d'avantage sur le langage JavaScript

    Framework PHP

    Un framework PHP est une bibliothèque de fonctions pré-écrites en PHP facilitant le développement d'applications fondées sur PHP. Le PHP permet de rendre les sites plus dynamiques.

    Le Framework Symphony est celui que j'etudirai le plus dans la deuxieme année de BTS

    Framework Python

    Un framework Python est une bibliothèque de fonctions pré-écrites en Python facilitant le développement d'applications fondées sur Python.

    Django est un framework open source développé en 2003. Il a pour but de rendre le développement d'applications web simple et basé sur la réutilisation de code.
